# ZMK config for phuhuynh Sofle V2

This config is based on zmk official docs with board nrf52840 module as controller

Docs is [here](https://zmk.dev/docs/user-setup)


Sofle vendor is Kiwikey.vn - documentation is [here](https://kiwikey.vn/zmk-keyboard-user-guide/)

Example config from ergomechstore [here](https://github.com/ergomechstore/sofle-v2-oled-zmk)

Keyboard editor for either connect to github repo or edit local file [here](https://nickcoutsos.github.io/keymap-editor/)

For Oled custom, using https://github.com/mctechnology17/zmk-nice-oled
